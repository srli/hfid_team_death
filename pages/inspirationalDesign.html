IMenuPro

Gabe Butterick

Imenupro is a website designed to create menus for restaurants in which you can pick which foods and categories should be on the menu. It has a similar ‘drag and drop from a pool, to what you want’ method we were thinking of using for the travel app. It lets the user create headers for activities to go under. Though it generates menus instead of travel itineraries, the idea is roughly the same if you replace meals with destinations and activities.

What’s good about it is:

  • It allows easy list building after initial setup time
  • Aesthetic is nice
  • Categorization
  • Demo with one click

What’s bad about it is:

  • Menu items must be entered in manually by the user
  • Categories are also not automatic
  • Requires initial setup
  • Requires signup
