Automated Prototype -- Invision App

Link to our Invision Prototype: LEIFY

Our app is relatively complex, which means we couldn't have new screens for EVERY option you want to click. Sorry. So, we have a script / set of steps we would like you to go through in order to test our app. In invision, not all of the buttons will work, but in theory our script should only lead you to ones that do. Thanks!

Script / Instructions

For all open ended questions, please send us your answers so we can use your input to improve our design.

  1. Home Page
    • What is your first instinct on this page?
    • If you wanted to use your current location, how would you do that? Can you set your current location on the screen?
    • Please set the time for this trip.
    • Now you’re ready to create your trip. Go ahead.
  2. Location Selection Screens

    3. Please only interact with the locations that we suggest, the others do not have functionality yet

    • What is your first impression of this screen? What do you think the purpose is?
    • Let’s say that you wanted to find out more information about Golden Gate Park. How would you do that?
    • How would you dismiss the card?
    • Now, add the Golden Gate Park to your trip.
    • Can you find a different way to add the Golden Gate Bridge?
    • Alright, you have one hour left in the trip, but you don’t like any of the three options provided. What do you do?
    • Now there are 3 new options. Can you add the Painted Ladies to your trip?
    • Now you don’t want to go to the Painted Ladies any more. Can you delete that place and add Fisherman’s Wharf instead?
    • Now confirm your trip!
  3. Summary Screen
    • Say you want to know how to get to the Golden Gate Bridge. How would would you do that?
    • You don’t actually want to navigate to just here, so exit that popup.
    • Now that you’re done reviewing this screen, open up the navigation instructions for your entire trip.
    • If this was implemented, clicking Yes would export this trip as via points to Google maps. However, because this is a prototype, just click No for now.
  4. Sign in / Preferences
    • So far, you’ve gone through this whole process as a general user. Because you like this app, you want to set up an account and set preferences for the next trip you take. We would now like you to log in and set your preferences.
    • How would you create an account?
    • The suggested budget that’s already filled in is a bit high. Can you lower the budget?
    • Can you exit this screen?

THANK YOU!
